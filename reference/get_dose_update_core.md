# Core function to calculate the dose update number for a row in a regimen update data.frame

Core function to calculate the dose update number for a row in a regimen
update data.frame

## Usage

``` r
get_dose_update_core(row, regimen)
```

## Arguments

- row:

  a single row from a regimen_update data.frame

- regimen:

  PKPDsim regimen
